Visual Studio Online quickstart
Visual Studio Online provides managed, on-demand development environments that can be used for long-term projects, to quickly prototype a new feature, or for short-term tasks like reviewing pull requests. Work with environments from anywhere using Visual Studio Code, Visual Studio integrated development environment (in private preview), or the included browser-based editor. Many other features, including do-it-yourself remote development, are also coming soon. See how to use Visual Studio Online in Visual Studio Code, create a cloud-hosted environment, connect to it, run and debug the environment's application, and disconnect and delete the environment—all in this quickstart.
