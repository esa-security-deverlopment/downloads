New Developer Documentation

    We've been working on bringing Trello's developer documentation to developer.atlassian.com (DAC) and we're almost there! Because you're reading this, you are getting an early preview of the new documentation. It is live at https://developer.atlassian.com/cloud/trello. Go check it out!
   
    By utilizing DAC, we've been able to pick up some exciting new features. For example, in the REST API Reference section you'll see a new button prompting you to "Download Postman Collection." You can download the entire REST API specification as a Postman collection or the entire OpenAPI Spec. This enables you to generate SDKs for your language of choice.

    We'd love your feedback, in general, on the new documentation. Head on over to the Trello developer community post and let us know what you think.

New Power-Up Metrics

    We've added a new section to the Power-Ups admin titled Metrics that displays a graph of your Power-Ups enablements on boards over time. The data is aggregated every 24 hours.

    Have ideas for additional metrics? Let us know what we should add next in the Trello developer community.
